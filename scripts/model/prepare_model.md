# Prepare Palace Model

1. Download the Palace of Fine Arts model from Sketchfab with a compatible license.
2. Keep attribution metadata (author, URL, license).
3. Convert to `.glb` if needed (Blender export or `gltf-transform copy`).
4. Place result at `data/models/palace.glb`.
5. For web serving, copy to `apps/web/public/models/palace.glb`.
6. Validate mesh names in the web debug panel before defining final hotspot mesh mappings.
